## The SRE Mindset
1. **Triage by impact** — what's causing the most user harm?
2. **Fix dependencies first** — Secrets before Deployments, etc.
3. **Verify each fix** — don't fix 5 things then check once
4. **Document as you go** — write the incident post-mortem
5. **Prevent recurrence** — add monitoring, alerts, and tests
